# gonpunclaw-population-decline-realestate

인구감소지역 부동산의 **실거래가**, **정책 특례**, **지역 맥락**을 함께 보여주는 공공형 안내 프로젝트.

## 왜 이 프로젝트인가

대부분의 부동산 서비스와 바이브코딩 프로젝트는 서울·수도권, 특히 관심이 집중된 고가 아파트를 중심으로 움직입니다. 
하지만 실제로 정보 비대칭이 더 큰 곳은 **인구감소지역**입니다.

- 정책 특례는 발표되지만
- 실제 지역별 부동산 정보는 파편화되어 있고
- 실수요자가 바로 이해할 수 있는 인터페이스는 부족합니다.

이 프로젝트는 그 간극을 메우는 것을 목표로 합니다.

## 핵심 기능(MVP)

- 인구감소지역 / 인구감소관심지역 목록 제공
- 시군구 단위 부동산 실거래가 조회
- 최근 거래 건수, 평균가, 중위가 제공
- 세컨드홈 특례 등 정책 요약 연결
- 법령/보도자료/공공데이터 링크 연결
- **특례 → 근거 법령 → 실제 거래 데이터** 흐름의 첫 화면 제공

## 데이터 소스

- 국토교통부 실거래가 API
- 행정안전부 인구감소지역/관심지역 자료
- 국토교통부 보도자료
- 법제처 국가법령정보 API
- 필요시 KOSIS 및 관련 공공데이터

## 방향 원칙

- 매물 중개 사이트가 아니라 **공공형 안내 사이트**로 간다.
- 투자 선동보다 **정보 비대칭 해소**에 초점을 둔다.
- 정책과 시장 데이터를 함께 보여준다.
- GitHub Pages 기반 정적 서비스로 시작한다.
