# 인구감소지역 부동산 — 특례와 실제 거래를 함께 보는 안내서

**배포 URL:** https://hosungseo.github.io/gonpunclaw-population-decline-realestate/

## 왜 이 사이트를 만들었나

수도권 부동산에는 수많은 서비스와 관심이 몰리지만, 인구감소지역의 시장을 체계적으로 보여주는 곳은 많지 않습니다.

토스, 쿠팡, 당근마켓 같은 플랫폼 기업은 많은 사용자를 기반으로 서비스를 제공합니다. 플랫폼이 유지되려면 대규모 이용자가 필수적이므로, 이들은 자연스럽게 관심이 이미 많은 방향으로 서비스를 설계합니다. 인구감소지역처럼 관심이 적은 영역에는 민간 플랫폼이 들어갈 유인이 약합니다.

하지만 AI 시대에는 사정이 다릅니다. 개발에 필요한 자원, 시간, 전문성의 허들이 극단적으로 낮아지면서, 대규모 사용자 기반 없이도 의미 있는 서비스를 만들 수 있게 됐습니다. 많은 사람이 관심을 가지지 않는 사안에 대해서도 서비스를 만드는 것이 가능해진 것입니다.

그리고 그것은 정부가 해야 하는 일이기도 합니다. 시장이 외면하는 곳에 공공의 가치를 제공하는 것이 정부의 존재 이유이기 때문입니다.

이 사이트는 그 가능성을 실험하는 [공픈클로(gonpunclaw)](https://github.com/hosungseo) 프로젝트의 일부입니다.

## 이 사이트는 무엇인가

인구감소지역·인구감소관심지역의 **법적 특례**와 **실제 부동산 거래**를 함께 보여주는 공공형 부동산 안내 서비스입니다.

가격보다 먼저, 왜 이 지역이 정책적으로 중요한지부터 보여줍니다.

## 구조

### 챕터 1: 그래서 무엇이 달라지나
- 세제특례 / 정주지원 / 도시계획 특례 / 관심지역 제도
- 각 특례별 핵심 효과 + 적용 조건 + 주의점 + 한 문장 요약
- 법률상 근거 토글 + 관련 정책뉴스 연결
- 근거 법률: 인구감소지역 지원 특별법, 조세특례제한법, 지방세특례제한법

### 챕터 2: 정책 대상 지역의 시장은 어떻게 움직였나
- 전국 지도에서 인구감소지역 89개 + 인구감소관심지역 18개 = **107개 전체 탐색**
- 시도 확대 2단계 + 지역 리스트 보조
- 지역 클릭 시 패널: 최근 24개월 거래건수/중위가격 그래프 + 평균선 + 해석 문구
- **지역 비교 테이블**: 정렬 가능, 시도 필터, 클릭 시 패널 연동
- 데이터 출처 명시
