# 지역코드 매핑 계획

## 목적
실거래가 API의 `LAWD_CD`(법정동 시군구코드 5자리)와 인구감소지역/관심지역 목록을 연결한다.

## 현재 상태
- 인구감소지역 89개 JSON 확보
- 인구감소관심지역 18개 JSON 확보
- 통합 인덱스 `population-region-index.json` 생성 완료

## 다음 작업
1. 시군구 코드 원본 확보
   - 법정동코드 전체 자료 또는 시군구 코드 CSV/TXT
2. 시도/시군구 명칭 정규화
   - 예: 강원 ↔ 강원특별자치도, 전북 ↔ 전북특별자치도
   - 현재 `data/province-normalization.json` 초안 작성 완료
3. 통합 인덱스에 `lawdCd` 필드 추가
4. 시범 지역 3곳에 대해 실거래가 API 테스트
   - 현재 `data/pilot-regions.json` 초안 작성 완료
   - `data/pilot-region-codes.json`에 1차 검증 결과 반영 완료

## 주의
- 실거래가 API는 시군구 단위 5자리 코드 기준
- 특광역시의 구/군과 도 단위 시/군 명칭을 구분해야 함
- 행정구역 명칭 변경(예: 군위군) 반영 필요
