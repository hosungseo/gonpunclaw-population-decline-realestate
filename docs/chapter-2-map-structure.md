# 2번 챕터 구조안 — 정책 대상 지역 지도에서 실제 거래로

## 한 줄 정의
전국 지도에서 인구감소지역·인구감소관심지역만 활성화하고, 해당 지역을 클릭하면 최근 2년 거래 정보를 보여주는 탐색형 섹션.

## 왜 지도형이 맞는가
- 이 프로젝트의 핵심은 "전국 부동산"이 아니라 **정책으로 지정된 지역**이다.
- 리스트보다 지도가 정책 대상 지역의 공간적 분포를 더 직관적으로 보여준다.
- 사용자는 "어디가 대상인지"를 먼저 이해하고, 그다음 "그 지역 시장이 어떤지"를 보게 된다.

## 사용자 흐름
1. 전국 지도에서 정책 대상 지역 확인
2. 인구감소지역 / 인구감소관심지역만 hover, click 가능
3. 특정 지역 클릭
4. 오른쪽 또는 하단 패널에 최근 2년 거래 정보 표시
5. 지역 상세 페이지 또는 더보기로 이동

## 지도 레이어
### 기본 레이어
- 일반 지역: 비활성(흐리게 처리)
- 인구감소지역: 진한 컬러
- 인구감소관심지역: 보조 컬러

### hover 상태
- 해당 시군구 강조
- 지역명 툴팁 표시

### click 상태
- 선택 지역 고정 강조
- 상세 패널 오픈

## 클릭 후 패널 정보
### 1. 기본 정보
- 지역명
- 시도
- 지역 유형 배지 (인구감소지역 / 인구감소관심지역)

### 2. 거래 요약
- 최근 24개월 거래 건수
- 최근 24개월 중위가격
- 최근 거래가 범위
- 최근 거래월

### 3. 대표 거래 예시
- 대표 아파트명 3~5건
- 거래금액
- 전용면적
- 건축연도

### 4. 정책 요약
- 이 지역에 적용되는 핵심 특례 2~3개
- 근거 법률 링크

## 2번 챕터 제목 후보
- 어느 지역에서 실제 거래가 있었나
- 정책 대상 지역의 시장은 어떻게 움직였나
- 지도에서 보는 인구감소지역 부동산 거래

## MVP 구현 범위
### 우선 구현
- 전국 지도 표시
- 정책 대상 지역만 색칠
- 클릭 시 샘플 지역 데이터 패널 표시

### 나중 구현
- 검색
- 필터
- 거래건수 기준 정렬
- 지도 줌/팬
- 월별 시계열 전환

## 디자인 원칙
- 1번 챕터가 설명 중심이면, 2번 챕터는 탐색 중심이어야 한다.
- 지도 자체가 강한 시각 앵커가 되도록 하고, 보조 패널은 정보 밀도를 높인다.
- 카드 남발보다 지도 + 패널의 2단 구조를 유지한다.
- 투자 권유형 톤이 아니라 관찰·비교형 톤을 유지한다.

## 추천 문장
- "특례가 있는 지역은 전국 어디에 있고, 실제 시장은 어떻게 움직였을까"
- "정책 대상 지역만 골라 실제 거래 흐름을 본다"
- "지정된 지역을 눌러 최근 2년 거래를 확인하세요"
