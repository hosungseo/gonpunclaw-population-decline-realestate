# GitHub Pages 첫 화면 구조안

## Visual thesis
정책 안내문과 부동산 시세표의 중간 지점. 과장된 투자 사이트가 아니라, 조용하지만 신뢰감 있게 **특례 → 근거 → 실제 거래**를 읽게 만드는 공공형 인터페이스.

## Content plan
1. Hero — 인구감소지역/관심지역 부동산 특례의 존재를 한 문장으로 설명
2. Support — 어떤 특례가 있는지 4개 축으로 요약
3. Detail — 특례가 적용되는 실제 지역의 거래 데이터 예시 제시
4. Final CTA — 지역별 상세 탐색으로 연결

## Interaction thesis
- 상단 특례 배지가 순차적으로 등장
- 스크롤 시 법률 근거 섹션이 부드럽게 드러남
- 지역 카드 hover 시 거래가와 특례 정보가 교차 강조됨

---

## 섹션 1. Hero
### 목적
사용자가 이 사이트를 "실거래가 사이트"가 아니라 **특례가 있는 지역의 시장을 보여주는 사이트**로 이해하게 만든다.

### 핵심 카피
- 헤드라인: **인구감소지역에는 특례가 있고, 그 특례는 실제 거래와 연결된다**
- 서브카피: 인구감소지역·인구감소관심지역의 법적 특례와 실제 부동산 거래를 함께 보여주는 공공형 부동산 안내 서비스
- 보조문구: 실거래가, 세제특례, 정주지원, 도시계획 특례를 한 번에 확인

### CTA
- `특례 먼저 보기`
- `거래 데이터 보기`

---

## 섹션 2. 특례 요약
### 목적
복잡한 법률을 빠르게 이해시키기

### 4개 축
1. **주거특례**
   - 공공임대 우선 공급
   - 노후주택 개선 지원
   - 공유지 우선 매각·대부
2. **세제특례**
   - 양도세·종부세 특례
   - 취득세·재산세 지원 확대
3. **도시계획특례**
   - 건폐율·용적률 완화 가능
4. **정착지원**
   - 생활인구 확대
   - 청년·중장년 정착 지원

### 표시 방식
- 카드 남발 금지
- 4열 또는 2x2 텍스트 블록
- 각 블록 아래 `근거 법령 보기` 링크

---

## 섹션 3. 법적 근거
### 목적
정책 설명의 신뢰성 확보

### 구조
- 인구감소지역 지원 특별법
  - 제24조 주거·교통기반 확충
  - 제24조의2 국토계획법 특례
- 조세특례제한법
  - 인구감소지역 주택 취득 국세 특례
  - 인구감소관심지역 확대
- 지방세특례제한법
  - 인구감소지역 지원 확대
  - 미분양·빈집 관련 지방세 지원

### 표시 방식
- 법령명 → 한 줄 요약 → 왜 중요한지
- 조문 원문 전체보다 서비스용 요약 우선

---

## 섹션 4. 실제 거래는 어떤가
### 목적
정책이 추상적 설명에 머물지 않도록 실제 거래 데이터 연결

### 1차 노출 지역
- 해남군
- 영주시
- 경주시

### 카드 내용
- 지역명
- 지역 유형 배지 (인구감소지역 / 인구감소관심지역)
- 최근 거래 건수
- 최근 거래가 예시
- 대표 아파트명 1~2개
- `상세 보기`

### 핵심 문장 예시
- "이런 특례가 있는 해남군은 최근 이렇게 거래되고 있습니다."
- "관심지역으로 확대된 경주시는 실제 거래가 활발한 편입니다."

---

## 섹션 5. 지역 탐색 CTA
### 목적
사용자를 지역 탐색으로 넘기기

### 카피
- "특례만 보지 말고, 실제 지역을 비교해보세요"
- 버튼: `지역별로 보기`

---

## UX 원칙
- 정책 설명이 먼저, 시세는 그 다음
- 투자 선동형 문구 금지
- 법률 원문보다 사용자가 이해할 수 있는 요약 우선
- 특례 → 근거 → 실제 거래 순서 유지
- 첫 화면은 카드 모음이 아니라 하나의 논리 흐름처럼 구성
