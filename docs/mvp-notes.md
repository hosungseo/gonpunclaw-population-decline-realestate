# MVP 메모

## 1차 화면 후보 지역
- 해남군 (인구감소지역)
- 영주시 (인구감소지역)
- 경주시 (인구감소관심지역)

## 선정 이유
- 최근 실거래가 데이터가 실제로 존재함
- 지역 유형 비교가 가능함 (인구감소지역 vs 관심지역)
- 첫 화면 카드/상세 화면에 뿌리기 좋은 데이터 밀도 확보

## 1차 화면 구성
- 지역 카드 3개
- 최근 거래 건수
- 최근 거래가 예시 3~5건
- 지역 유형 배지 (인구감소지역 / 관심지역)
- 정책 특례 설명 링크
